# demotest

This repository contains a shell script for monitoring CPU and memory usage on a machine.

## cpu_monitor.sh

- A shell script that monitors CPU and memory usage every 5 seconds.
- Useful for tracking system performance over time.

## Usage

1. Clone the repository.
2. Run the `cpu_monitor.sh` script:  
   ```sh
   bash cpu_monitor.sh
   ```
3. The script will print CPU and memory usage every 5 seconds.
